# Coop Roadmap

## Stage 1: Foundation (done)
- Базовий модуль, build/deploy loop, стартові мережеві команди.

## Stage 2: Spectator base (done)
- Базова синхронізація стану і обмеження в campaign контексті.

## Stage 3: Battle integration (active)
- 3.1 Battle detection: done.
- 3.2 Dedicated/listed join bridge: done.
- 3.3 Unit selection + spawn + spectator transitions: in progress.
- 3.4 Server-authoritative combat loop: next.
- 3.5 Return battle outcome to campaign: next.

## Stage 4: Stability & UX
- Reconnect scenarios.
- Better user-facing diagnostics.
- Long-session reliability.

## Stage 5: Playtest & release
- 2/3/4-client matrix.
- Performance pass.
- Packaging and user documentation.

## Delivery rules
- Один ризиковий технічний блок за ітерацію.
- Кожна ітерація має test checklist.
- Після ітерації: короткий status update у `HUMAN_NOTES_MULTIPLAYER_PROGRESS.md`.
